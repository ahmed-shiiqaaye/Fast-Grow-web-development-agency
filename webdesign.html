what is the difference between a web designer and a graphic desinger

As mentioned above, web designers focus on the aesthetics of the website. Graphic designers on the other hand, create visual elements such as logos, business cards, invitations, flyers, etc. Therefore, graphic designers create design elements that can be used on a website whereas web designers focus on the design of the site itself.


WHAT IS MOBILE WEB DESIGN?

74% of all website visitors are more likely to visit a website if it is mobile-friendly.


Mobile web design ensures that your website can be viewed properly on a mobile device. Websites are designed using a desktop and because mobile devices have much smaller screens, sections of your website may appear jumbled, or may not appear on a mobile device at all.


Instead of using the term “mobile web design” web designers call this “responsiveness”. This means that your website is “responsive”, it can be viewed properly on desktop, mobile and tablet. More and more people are using their mobile devices when conducting searches, which is why responsiveness is crucial for your website.
